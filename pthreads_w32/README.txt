This library is necessary for binaries using OpenMP compiled with the MinGW-w64 compiler.  To prevent them from crashing, you should 
copy the pthreadGC2-w32.dll file to the same path as your executable(s).

It was compiled using MSYS by Michael Sarahan on 3-May-2011.

pthreads-win32 can be found here:
http://sourceware.org/pthreads-win32/
